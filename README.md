# Ademide Cinematic Portfolio

A 10-page portfolio site inspired by a cinematic product launch layout, translated into a professional software portfolio.

## Pages

- `index.html`
- `about.html`
- `work.html`
- `services.html`
- `stack.html`
- `case-studies.html`
- `process.html`
- `lab.html`
- `journal.html`
- `contact.html`

## Run locally

```bash
python serve.py
```

Then open `http://127.0.0.1:4321`.
